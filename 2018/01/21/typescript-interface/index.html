Typescript | Interface

2018-01-21 17:33:54
Dev>Typescript

소개

타입스크릡트 핵심원리 중 type-checking 은 값이 갖는 shape(type)에 초점을 맞추는 것이다.
이것을 duck typing, 이라고 한다.
이러한 유형의 이름을 지정하는 역할을 수행하며 코드 외부에서 코드를 사용하여 계약서를 정의하는 강력한 방법입니다.

파라메터 labelledObj: { label: string } 를 주목하자.

이제 이것을 interface 를 적용하면 다음과 같이 된다.

파라메터 labelledObj: { label: string } 대신 interface LabelledValue로 바뀌었다.


Optional Properties

타입스크립트에서 인터페이스는 구현하는 클래스에서 모두 온전히 override를 하지 않도록 optional 을 지원하고 있다.
