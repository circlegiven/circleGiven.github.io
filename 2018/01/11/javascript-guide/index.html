자바스크립트 가이드

2018-01-11 11:41:25
Dev>Javascript

이 글은 ‘자바스크립트 완벽가이드’ 와 ‘You don’t know JS’를 읽으며 정리한 글 이다.

이 페이지에서는 자바스크립트에 대한 기초 가이드만 다룰 것이며,

자바스크립트 원리 및 심화 같은 경우 그때마다 따로따로 포스팅을 쓸 계획이다.

또한 책을 끝까지 읽을 때까지 계속해서 업데이트 할 계획이다.

자바스크립트의 타입

자바스크립트는 다른언어(JAVA, C) 처럼 선언시 int, float, string 같은 것을 붙이지는 않지만
엄연히 타입이 존재한다.

  • null
  • undefined
  • boolean
  • number
  • string
  • object
  • symbol (ES6 부터 추가)


타입은 typeof 로 확인할수 있다.


null은 object 라고 표시되는데 이건 자바스크립트의 특성이다.
