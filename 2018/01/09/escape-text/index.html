문자열 내에서 줄바꿈 사용하기


자바스크립트의 문자열 내에서 줄바꿈을 사용하려면 이스케이프 시퀸스 라는것을 사용해야 한다.

시퀸스 표현하는 문자
\0 (\u0000) null 문자
\b (\u0008) 역스페이스
\t (\u0009) 수평 탭
\n (\u000A) 줄바꿈 문자
\v (\u000B) 수직 탭
\f (\u000C) 폼 피드
\r (\u000D) 캐리지 리턴
\” (\u0022) 큰따음표
\’ (\u0027) 작은따음표
\ (\u005C) 역슬래시
\x 두 개의 16진수 숫자
\u 네 개의 16진수 숫자



문자열 내에서 사용은 이런식으로 한다.

유니코드로도 가능하다
